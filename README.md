# AI-Powered Indian Sign Language to Speech System

This project is an AI-based system that recognizes Indian Sign Language (ISL) gestures using a webcam and converts them into speech in real time. The system helps bridge communication between deaf or hard-of-hearing individuals and others.

## Features
- Real-time hand gesture recognition
- Conversion of sign language gestures to text and speech
- Webcam-based gesture detection
- AI-based gesture classification

## Technologies Used
- Python
- MediaPipe
- LSTM (Deep Learning Model)
- React (Frontend)
- Flask (Backend)
- Text-to-Speech

## How It Works
1. The webcam captures hand gestures.
2. MediaPipe detects hand landmarks.
3. The LSTM model classifies the gesture.
4. The recognized gesture is converted into text.
5. Text-to-speech generates the corresponding audio output.
